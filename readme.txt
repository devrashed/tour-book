Very simple way you can make your Tour book by the plugin. 

== Description ==

This plugin enables users to select their desired destination, tour cost, and duration to confirm a tour with ease. No technical expertise is required to operate the plugin. Simply insert the **[tour_filter_form]** shortcode into any page or post to display the tour booking form and activate its features.
